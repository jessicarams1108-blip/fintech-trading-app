================================================================================
  HOW TO PREVIEW YOUR WORK AFTER EVERY UPDATE (simple version)
================================================================================

WHERE IS YOUR PROJECT?
  Folder:  fintech-trading-app
  (Full path might be like: C:\Users\YOURNAME\fintech-trading-app)


EVERY TIME YOU WANT TO SEE YOUR SITE IN THE BROWSER
--------------------------------------------------------------------------------

  1) Save your files in Cursor (Ctrl+S).

  2) Go to the project folder on your computer (fintech-trading-app).

  3) Double-click this file:
       PREVIEW-MY-WORK.cmd

  4) A black window opens — DO NOT CLOSE IT while you preview.
     Wait until you see text like "Local: http://localhost:5173/"

  5) Your browser should open automatically to:
       http://localhost:5173/

     If it does not open: copy that address, paste into Chrome/Edge, Enter.

  6) When you change code: save the file, then in the browser press:
       Ctrl + Shift + R   (hard refresh)
     Or click refresh.

  7) When you are done: close the black window (or press Ctrl+C in it).


IF YOU SEE "NODE.JS WAS NOT FOUND" (black window, then "Press any key")
--------------------------------------------------------------------------------
  Windows did not find Node when you double-clicked the file.

  EASIEST FIX — use Cursor's terminal (Node usually works there):
    1. Open your project in Cursor
    2. Terminal menu -> New Terminal
    3. Type:  cd   then drag your fintech-trading-app folder into the terminal
              (or type the full path to that folder)
    4. Press Enter, then type:  npm run dev -w web
    5. Open browser: http://localhost:5173/start

  OR install / repair Node from https://nodejs.org (LTS) and tick "Add to PATH",
  then restart the PC and double-click PREVIEW-MY-WORK.cmd again.


IF THE BROWSER SAYS "CONNECTION REFUSED"
--------------------------------------------------------------------------------
  That means step 3 is not running or you closed the black window.
  Fix: double-click PREVIEW-MY-WORK.cmd again and keep the window open.


IF YOU SEE ERRORS ABOUT /api (deposit, login)
--------------------------------------------------------------------------------
  The website still shows, but backend is off.
  Optional: double-click start-server.cmd in a SECOND window
  (needs .env file — see QUICK-START.txt).


THAT'S IT
--------------------------------------------------------------------------------
  Same steps every time:  save → double-click PREVIEW-MY-WORK.cmd → browser.
