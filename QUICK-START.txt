FASTEST PREVIEW (frontend only, after every code change):
   Double-click:  PREVIEW-MY-WORK.cmd
   Read:          HOW-TO-PREVIEW.txt

---

1) Create your env file:
   copy .env.example .env

2) Edit .env and set:
   - DATABASE_URL
   - JWT_SECRET

3) Start both backend + frontend:
   start-all.cmd

4) Open:
   http://localhost:5173/

If you only start frontend, the UI still works. Login and saving deposits need the backend
(start-all.cmd or npm run dev:server + .env).
